For power users who want a clean way to manage Claude Desktop without editing JSON.

Toggle MCPs instantly

One checkbox. The app updates your config and you are done. No JSON, no typos.

See what is running

Live status indicator for each MCP. Spot zombies before they freeze your session.

Import in seconds

Paste JSON, drop a file path, or clone from Git. Works for MCPs and Skills.

Auto-update from GitHub

New release published? The app shows a badge. One click and you are on the latest version.

100% local

Runs on localhost:8765. No telemetry, no analytics, no cloud. Your config stays on your Mac.

Zero dependencies

Pure Python stdlib. No pip install, no node_modules. Just download and run.

Install in 30 seconds

macOS only. Requires git + python3 (preinstalled on macOS).

curl -fsSL https://claude.sekoia.ca/install.sh | bash

After install, double-click Claude Control.app on your Desktop.
